Running the Project

Dataset: 
The dataset used for this project can be obtained from the following site:
https://www.kaggle.com/rounakbanik/the-movies-dataset

Libraries to install:
Note that this project runs using Python 3.6, thus please install the compatible version of these modules. 
Scrapy, Pandas, Scikit-Learn, Crochet

Instructions:
The program automatically runs after gui.py is compiled. Due to the amount of data, expect it to take 2-2.5 minutes to compile.
